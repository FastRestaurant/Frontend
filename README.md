<div align="center">
  <h1>🍽️ Frontend</h1>
  <h3> Interfaz web encargada de la experiencia de usuario de la plataforma. </h3>
</div>

* **Panel de Administración:** Gestión de platos, bebidas, stock, mesas, usuarios y configuración general del restaurante.
* **Panel de Mozo (Waiter):** Manejo de mesas, toma de pedidos y seguimiento de operaciones en tiempo real.
* **Panel de Cocina (Kitchen):** Cola de pedidos en vivo para la preparación y despacho de platos.
* **Autenticación:** Inicio de sesión integrado con el microservicio de autenticación mediante JWT.
* **Notificaciones en Tiempo Real:** Actualizaciones instantáneas de pedidos y mesas vía SignalR.
